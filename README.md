## Getting started

```bash
# Clone this repo using
$ git clone https://github.com/wellingtondw/moveIt.git

# Move yourself to the appropriate directory
$ cd moveIt

# Run yarn to install dependencies
$ yarn
```

<br>

## Starting project

```bash
# Run yarn start to start project
$ yarn start
```
